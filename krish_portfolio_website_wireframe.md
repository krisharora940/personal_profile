# Krish Arora — Portfolio Website Wireframe

## Site Positioning
**Krish Arora**  
*Math student building AI/ML systems, trading research tools, and technical software.*

This site should present Krish as an emerging **research-minded engineer**, not as a generic student and not as a freelance web designer.

## Core Design Direction
- Premium, restrained, technical
- Dark/light/system theme toggle
- Strong typography, large spacing, minimal noise
- Quiet animations and polished hover states
- Institutional-research feel with modern product quality
- Charts and metrics should feel Bloomberg / research dashboard, not retail-finance flashy

## Primary Goals
1. Showcase flagship projects with serious case-study presentation
2. Highlight quantitative and technical credibility
3. Create room for future ML papers, notes, and experiments
4. Give a clean narrative about background, interests, and trajectory
5. Allow easy insertion of screenshots, charts, and GitHub links later

# Component Notes for Developer
- Build chart containers so Krish can swap in actual screenshots later
- Also support native rendered charts if data becomes available
- Use polished skeleton / empty states for unfinished research sections
- Make project cards reusable for future additions
- Preserve strong editorial spacing and typography
- Avoid overcrowding the homepage
- The homepage should feel like a selective summary, not a full archive

---

# Copy Style Rules
- Avoid hype words like visionary, disruptive, passionate, obsessed
- Tone should be direct, technical, credible
- Keep blurbs short and controlled
- Make unfinished sections feel intentional and forward-looking

- a serious technical portfolio
